# ❤️ Heart Disease Risk Analysis and Prediction

**Course:** Data Science and Big Data Analytics (DSBDA)

---

## 📌 About the Project

This project focuses on analyzing heart disease risk factors and building a machine learning model to predict the likelihood of heart disease in individuals. It also includes interactive data visualizations using Power BI.

---

## 📁 Project Structure

| File | Description |
|------|-------------|
| `Heast_Disease_Responses.csv` | Raw dataset collected (survey/patient responses) |
| `cleaned_heart_data.csv` | Preprocessed and cleaned dataset |
| `cleaning.ipynb` | Jupyter Notebook for data cleaning and preprocessing |
| `Heart Disease Risk Prediction.ipynb` | Jupyter Notebook for ML model building and evaluation |
| `Heart_disease_risk_analysis.pbix` | Power BI dashboard for visualization |

---

## 🔄 Workflow

1. **Data Collection** — Raw heart disease response data in CSV format
2. **Data Cleaning** — Handling missing values, encoding, and normalization 
3. **Model Creation** — Training and evaluating a classification model
4. **Visualization** — Interactive Power BI dashboard for risk analysis 

---

## 🧠 ML Model

- **Task:** Binary Classification (Heart Disease: Yes / No)
- **Algorithm:** Logistic Regression, KNN, Random Forest Classifier
- **Evaluation Metrics:** Accuracy, Confusion Matrix, Classification Report

---

## 📊 Visualizations

The Power BI report (`Heart_disease_risk_analysis.pbix`) includes:
- Risk factor distribution charts
- Age vs. heart disease correlation
- Feature-wise analysis dashboards

---

## 🛠️ Technologies Used

- Python (Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn)
- Jupyter Notebook
- Power BI
- CSV (Dataset)

## ▶️ How to Run

1. Clone the repository
   ```bash
   git clone https://github.com/RupaliShewale09/DSBDA-Heart-Disease-Risk-Analysis-Prediction.git
   ```

3. Open `Heart Disease Risk Prediction.ipynb` and run all cells to train and evaluate the model
